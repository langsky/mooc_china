## Python语法元素分析

### 程序元素
注释，缩进，变量，常量，表达式，输入，输出，分支，循环

```python
# TempConvert.py
val = input("请输入带温度表示符号的温度值（例如：32C）：")
if val[-1] in ['c', 'C']:
    f = 1.8*float(val[0:-1])+32
    print("转换后的温度为 %.2fF"%f)
elif val[-1] in ['f',"F"]:
    c = (float(val[0:-1])-32)/1.8
    print("转换后的温度为 %.2fC"%c)
else:
    print("输入有误")
```

#### 缩进
1. Python的一个缩进等于四个空格。
2. 缩进用来表示代码的层级关系。
3. 缩进是Python用来表明程序框架的唯一手段。

#### 常量和变量
1. 在Python中，常量和变量的使用没有基本区别。
2. 命名规则，可以使用非英文。
3. 命名规则，不要使用保留字。
