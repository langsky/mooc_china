## 简单分支结构

简单分支结构就是选择结构。

简单条件就是决策条件。简单条件就是一个布尔表达式。

二分支决策就是if-else语句。

## 多分支

多分支就是if-elif-else语句。
多分支决策可以由双分支嵌套得到。

## 异常处理

异常处理是用来捕捉运行时错误的语句。可以捕捉任何类型的错误。

```python
try：
    ...
except <error>:
    ...
else:
    ...
finally:
    ...
```

try没有异常就执行else。finally是必须执行的。
